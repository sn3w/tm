A hybrid kernel based minimal OS aimed at providing better experience to end-users of 9Front.
It allows better API support, extensibility, middleware support, driver hackability, and favourable
ring design.

Provided that it implies to the following rules:
    -> Security: Should be secure and must have some mitigation to exploitation
    -> Minimal: approach and Simplistic design: Approach must be done in the most optimal and simplest way
    -> Performant: Performance is necessary as security and minimalism cannot be built on the torned down foundation of performance
